*RegexMarkup* Develop
=====================
**RegexMarkup** is based on regular expressions rules to identify patterns in the references of journals, for identify most elements of one reference. The patterns are stored in an xml file, which defines one or more rules for review. For application development using `VSTO (Visual Studio Tools for Office) <http://msdn.microsoft.com/en-us/office/hh133430>`_ and as main programming language C#.
